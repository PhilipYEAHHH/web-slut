# ✅ TodoListAPI

A simple REST API built with **Node.js, Express, and SQLite** for managing a list of todos.  
Supports full CRUD functionality with input validation. Includes a simple HTML/CSS/JS frontend.

---

## 🚀 Features

* Get all todos  
* Get a todo by ID  
* Add a new todo  
* Update an existing todo  
* Delete a todo  
* Mark todos as completed  
* Built-in validation for all inputs  
* SQLite database (auto-created)  
* Simple frontend to view and manage todos in the browser  

---

## 🛠️ Tech Stack

* Node.js  
* Express  
* SQLite3  
* HTML/CSS/JavaScript (frontend)
